# ToDoList

This project is an SPA for storing and managing events and tasks.

## Features

- Google authorization
- Firestore database
- Up/download tasklist
- Multiple sorting and filtration

## Tech stack

- Angular 5
- Fontawesome 4.7
- Bootstrap 4.0
